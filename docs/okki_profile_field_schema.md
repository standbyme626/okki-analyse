## 字段用途分类

### identity_fields
- 公司名称
- 客户编号
- 国家地区
- 公司网址
- 座机
- 客户代码

### scoring_fields
- 客户来源
- 客户阶段
- 客户等级
- 年采购额
- 采购意向
- 客户销售渠道
- 最近联系时间（来自跟进信息）

### writable_candidates
- 客户等级
- 客户标签（头部标签区，不在本两分组内）

### read_only_evidence_fields
- 时区
- 规模
- 产品分组
- 详细地址
- 公司备注
- 公司logo
- 传真
- 询盘产品
