# OKKI Action Catalog (Recon / No Write)

## detect_page_mode()
- 目标: 判断当前页面属于 `drawer` / `full_page` / `unknown`
- 适用页面模式: both
- 前置条件: 已获取当前 URL + title + `snapshot -i`
- 稳定定位策略: URL 模式匹配（`/customer/personal`）、标题文本、是否存在右侧抽屉+列表共存结构
- 不可使用策略: 固定 `@eXX`、固定坐标、依赖一次性 DOM 层级索引
- 失败条件: URL/title/snapshot 互相矛盾或缺失
- 需要截图节点: `before-read`
- 是否允许 dry_run: 是（只读）
- 是否涉及真实写入: 否

## read_customer_name()
- 目标: 读取客户名称
- 适用页面模式: both
- 前置条件: 已识别 page_mode；详情区域已可见
- 稳定定位策略: `full_page` 用主标题 heading；`drawer` 用抽屉标题区域中的名称文本
- 不可使用策略: 固定 ref、坐标点击获取
- 失败条件: 标题区未渲染/名称为空
- 需要截图节点: `before-read`, `on-error`
- 是否允许 dry_run: 是
- 是否涉及真实写入: 否

## read_customer_level()
- 目标: 读取客户等级当前值
- 适用页面模式: both
- 前置条件: `资料`上下文可读（已选中或已展示）
- 稳定定位策略: 语义锚点 `客户等级` 标签 + 邻近值节点
- 不可使用策略: 通过编辑控件值反推、固定 index
- 失败条件: 仅找到标签找不到值；字段被折叠/权限不可见
- 需要截图节点: `before-read`, `on-error`
- 是否允许 dry_run: 是
- 是否涉及真实写入: 否

## read_customer_tags()

- 适用页面模式: both

- 不可使用策略: 假设标签固定在某个 ref
- 失败条件: 标签区空渲染或异步未加载
- 需要截图节点: `before-read`, `on-error`
- 是否允许 dry_run: 是
- 是否涉及真实写入: 否

## open_customer_detail_from_list()
- 目标: 从列表打开客户详情
- 适用页面模式: drawer
- 前置条件: 当前在客户列表/搜索结果页，客户行可见
- 稳定定位策略: 客户名文本 link/row 语义定位；打开后再判定是 `drawer` 还是 `full_page`
- 不可使用策略: 固定第 N 行、固定坐标、固定 ref
- 失败条件: 点击后无导航且无抽屉出现
- 需要截图节点: `before-read`, `on-error`

- 是否涉及真实写入: 否

## search_customer()
- 目标: 在列表页定位目标客户
- 适用页面模式: drawer
- 前置条件: 客户列表页存在搜索输入框
- 稳定定位策略: 输入框占位符/label（`请输入搜索关键字`）+ 查询后验证列表中客户标识
- 不可使用策略: 写死 ref、强依赖单一 class hash
- 失败条件: 无结果、输入框不可交互
- 需要截图节点: `before-read`, `on-error`
- 是否允许 dry_run: 是
- 是否涉及真实写入: 否

- 需要截图节点: `before-read`, `on-error`

## verify_after_write()
- 目标: 写入后回读验证（本轮只设计，不执行真实写入）
- 适用页面模式: both
- 前置条件: 已有预期值（目标等级/标签）
- 稳定定位策略: 重新读取同一语义字段并与期望值比较
- 不可使用策略: 仅靠 toast 成功提示判断
- 失败条件: 读取失败、值不一致、页面刷新后丢失
- 需要截图节点: `after-write`, `on-error`
- 是否允许 dry_run: 是（可用模拟数据）
- 是否涉及真实写入: 否（本轮）
